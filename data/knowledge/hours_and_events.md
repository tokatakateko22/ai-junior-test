# NovaBite Opening Hours, Branch Rules, and Events

## Branches
- Downtown
- Riverside
- Tech Park

## Opening Hours

### Weekdays (Monday to Friday)
- Downtown: 09:00 to 22:00
- Riverside: 10:00 to 22:00
- Tech Park: 08:00 to 21:00

### Weekends (Saturday and Sunday)
- Downtown: 10:00 to 23:00
- Riverside: 10:00 to 23:00
- Tech Park: 09:00 to 22:00

## Branch Policies
- Last kitchen order is 30 minutes before closing.
- Reservations are held for 15 minutes.
- For groups above 8 guests, a pre-order may be required.

## Event Hosting
- Birthday events are available in Downtown and Riverside branches.
- Standard birthday package includes:
  - Reserved seating for up to 20 guests
  - Custom menu cards
  - Dessert platter
- Premium birthday package includes:
  - Everything in standard package
  - Dedicated host
  - Decor setup
  - Audio support
- Tech Park branch does not offer private event hosting on weekends.
